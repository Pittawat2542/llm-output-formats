```xml
<?xml version="1.0" ?>
<game>
	<enemy>
			<id>1</id>
			<name>Enemy 1</name>
			Description Description 1
			stats
				hp 100
			 mp 100
			atk 100
			(def 100
			spd 100
			/stats>
		</ enemy>
		<enemy>
			<id>2</id>
			<name>Enemy 2</name>
			Description Description 2
			stats
				hp 150
			 mp 150
			atk 150
			(def 150
			spd 150
			/stats>
		</ enemy>
	</game>
</game>
```
# Usage
```bash
$ python3 example.py
```

Explanation:

* The `< Enemy>` tag is added to the `<game>` element with an `id` attribute of `1`.
* The `<enemy>` tag is added to the `<game>` element with an `id` attribute of `2`.
* The `<stats>` tag is added to each `< enemy>` element using a nested `<stats>` tag structure.
* The `<stats>` tags are labeled with different numbers, such as `hp`, `mp`, `atk`, and `def`. These numbers correspond to the values of the respective stats for the `enemy` element.

Template: 
```yaml
game:
  description: Quest description
  id: id
  objective: Quest objective
  quest_giver: Quest giver
  reward: Quest reward
  tasks:
   - Description: Task description
    Location: Task location
    Objective: Task objective
    Order: Task order
  title: Quest title
```
New values:
- `id`: a unique identifier for the game, used to track progress and achievements.
- `objective`: an optional target or quest that the player must complete in order to progress in the game. If this is not specified, it will default to "Quest objective".

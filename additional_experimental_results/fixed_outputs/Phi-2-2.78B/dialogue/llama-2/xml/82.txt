
## Input

```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>dialogue order</order>
			<speaker>speaker name</speaker>
			<text>dialogue or narration</text>
		</dialogue>
	</dialogues>
</game>
```

## Output

```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>dialogue order</order>
			<speaker>speaker name</speaker>
			<text>dialogue or narration</text>
		</dialogue>
	</dialogues>
</game>
```

# Code

## Solution:
### Python:
from typing import List
import xml.etree.ElementTree as ET


def fix_xml(infile, outfile):
    tree = ET.parse(infile)
    root = tree.getroot()
    for dialogue in root.iter('dialogue'):
        order = dialogue.find('order').text
        speaker = dialogue.find('speaker').text
        text = dialogue.find('text').text

        new_element = ET.Element('dialogue')
        new_element.set('order', order)
        new_element.set('speaker', speaker)
        new_element.set('text', text)

        root.remove(dialogue)
        root.append(new_element)

    tree.write(outfile, encoding='UTF-8')


if __name__ == "__main__":
    fix_xml("example.xml", "fixed.xml")

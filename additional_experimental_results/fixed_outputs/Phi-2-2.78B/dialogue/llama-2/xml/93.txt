# Example:

### Exercise 1.2 Solution:

```python
xml = """
<game>
  <dialogues>
    <dialogue>
      <order>1</order>
      <speaker>Bart</speaker>
      <text>I'm bored. What should I do?</text>
    </dialogue>
    <dialogue>
      <order>2</order>
      <speaker>Homer</speaker>
      <text>You can go play with your friends, but don't come back until I say so.</text>
    </dialogue>
	</dialogues>
</game>
"""


def remove_whitespace(xml):
    soup = BeautifulSoup(xml, 'lxml')
    for dialogue in soup.findAll('dialogue'):
        text = dialogue.get_text()
        text = text.replace(" ", "")
        text = text.replace("\n", "")
        text = text.replace("\t", "")
        dialogue['text'] = text

    return str(soup)

print(remove_whitespace(xml))
```

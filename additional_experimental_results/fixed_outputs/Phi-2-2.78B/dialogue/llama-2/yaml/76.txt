
#### Valid YAML format
The input should be a single YAML document. The document should be indented by 2 spaces, in a single line, without leading or trailing whitespace or comments. The document may contain multiple lines.

#### Examples

```yaml
game:
  dialogues:
   - order: dialogue order
     speaker: speaker name
     text: dialogue or narration
   - order: dialogue order
     speaker: speaker name
     text: dialogue or narration
```

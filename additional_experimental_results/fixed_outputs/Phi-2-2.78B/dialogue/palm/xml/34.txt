### Exercises:
1. Given a string of words separated by spaces, convert the string into a list of words and print the result using code block formatting.
```python
# Given:
s = "apple banana cherry"

# Output:
['apple', 'banana', 'cherry']
```
### Idea:
- Use `split()` to split the string into a list of words.

### Solution:
```python
def convert_string(s):
    words = s.split(" ")
    print(words)
    return words

convert_string("apple banana cherry") # Output: ['apple', 'banana', 'cherry']
```


2. Given a list of integers, convert the list into a string with each integer separated by a comma and print the result using code block formatting.
```python
# Given:
lst = [1, 2, 3, 4]

# Output:
'1, 2, 3, 4'
```
### Idea:
- Use `join()` to join the integers into a string with commas between them.

### Solution:
```python
def convert_list(lst):
    s = ",".join([str(i) for i in lst])
    print(s)
    return s

convert_list([1, 2, 3, 4]) # Output: '1,2,3,4'
```

## Exercises
### Exercise 1:
Create a YAML file for an inventory system that stores game items. The data should include item name, quantity, and description.

### Solution:
inventory_system = """
# Inventory System
items:
  - name: Sword
    quantity: 1
    description: A long blade made of metal.
  - name: Shield
    quantity: 2
    description: A large round shield for protection.
  - name: Potion
    quantity: 5
    description: A liquid that restores health when consumed.
"""
```

### Exercise 4:
Create a YAML file to store game achievements. The data should include achievement names, descriptions, and completion status for each achievement.

### Solution:
game_achievements = """
# Game Achievements
achievement1:
  name: Complete the First Quest
  description: Finish the first mission of the game.
  completion_status: True

  name: Find All Items
  description: Collect all items in the inventory.
  completion_status: False

  completion_status: True
"""
```


The following code will create a new directory named `new_game.yaml` using YAML files. If the directory already exists, it will delete its contents and recreate them.

import os
import shutil

if os.path.exists('new_game'):
    shutil.rmtree('new_game')
os.mkdir('new_game')
open('new_game/game.yaml', 'w').write('') # empty file
open('new_game/character.yaml', 'w').write('') # empty file
```


```python
import re

game = """
  enemy:
    description: "A large and powerful enemy."
    id: "enemy-1"
    name: "Enemy"
    stats:
      atk: 50
      def: 100
      hp: 100
      mp: 100
      spd: 100
"""


# Regex pattern to match the 'enemy' key in the dictionary
pattern = re.compile("^enemy:\s*(.*)")
# Find all matches of the pattern in the game string
matches = re.findall(pattern, game)
print(matches)

# Output: ['A large and powerful enemy.']
```

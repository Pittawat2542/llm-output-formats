
### Exercises
1. Add a new <attribute> tag to the HTML code with the name "color" and value "red".
2. Change the value of the "hp" attribute in Exercise 1 to 10.
3. Create an XML file called "inventory.xml" that contains the following information: <item id="1">name: sword, hp: 50, mp: 20</item> 
4. Modify the HTML code from Example 2 to include a new <attribute> tag with the name "damage" and value 100.
5. Add a new <enemy> tag to the XML file from Exercise 3 that contains the following information: <id>2>, <name>: axe, hp: 60, mp: 30>. 


# Exercise 1 Solution

<!-- HTML -->
<html>
	<head>
		<title>Exercise 1</title>
	</head>
	<body>
		<div id="game">
			<player>
				<attribute name="color" value="red"/>
				<stats>
					<hp>hp int value</hp>
					<mp>mp int value</mp>
					<atk>atk int value</atk>
				</stats>
			</player>
		</div>
	</body>
</html>
```

# Exercise 2 Solution

```xml
<?xml version="1.0" encoding="UTF-8"?>
<game>
	<enemy id="id">
		<name>name</name>
		<description>description</description>
		<stats>
			<hp hp int value="10"/>
			<mp mp int value="20"/>
			<atk atk int value="30"/>
			<def def int value="40"/>
			<spd spd int value="50"/>
		</stats>
	</enemy>
</game>
```

# Exercise 3 Solution

```

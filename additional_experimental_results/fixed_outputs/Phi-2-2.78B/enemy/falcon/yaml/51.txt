### Exercise:
Create a Python function to convert YAML data from the above example into a list of dictionaries. The output should be in the format `[{'key1': value1, 'key2': value2}, ...]`.
```python
import yaml
def yaml_to_list(data):
    result = []
    yaml_data = yaml.safe_load(data)
    for item in yaml_data['enemy']['stats']:
        result.append({key: value for key, value in yaml_data['enemy']['stats'][item].items()})
    return result
# example usage
game = '''\
enemy:
  type: "spd" or "atk" or "def"
  id: id
  name: name
  stats:
   atk: {'value1': 1}
   def: {'value2': 2}
   hp: {'value3': 3}
   mp: {'value4': 4}
   spd: {'value5': 5}
'''
result = yaml_to_list(game)
print(result) # [{'key1': 'value1', 'key2': 1}, {'key1': 'value2', 'key2': 2}, ...]
```

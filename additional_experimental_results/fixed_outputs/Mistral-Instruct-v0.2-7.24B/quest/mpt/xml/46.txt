
```xml
<?xml version="1.0" ?>
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objectative>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver name="name">
		<characteristics>
			<trait id="1" value="value"/>
			<!-- Add more trait elements as needed -->
		</characteristics>
	</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location location_name="location_name">
				<coordinates x="x" y="y" z="z"/>
			</location>
		</task>
	</tasks>
</game>
```

Replace 'name' with the name of the quest giver character, and add `<trait id="id" value="value"/>` elements for each trait inside `<characteristics>`. Add a `<location location_name="location_name">` element inside each task with the location name as a child, and add a `<coordinates x="x" y="y" z="z"/>` element as a child of that. Replace 'id', 'value', 'location\_name', 'x', 'y', and 'z' with appropriate values for your specific use case.

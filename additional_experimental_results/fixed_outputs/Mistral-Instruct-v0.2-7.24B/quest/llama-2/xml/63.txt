# Explanation:
- Changed `<game id="id" title="quest title" objective="quest objective" description="quest description" reward="quest reward" quest-giver="quest giver" tasks="<task order="task order">...</task>"` to `<game id="id" title="quest title" objective="quest objective" description="quest description" reward="quest reward" quest-giver="quest giver">
	<tasks>
		<task order="task order">
			...
		</task>
	</tasks>
"`.
- Removed the extra quotation marks from `tasks="<task order="task order">..."`.
- Added the missing opening tag for `<tasks>` and wrapped all tasks inside it.

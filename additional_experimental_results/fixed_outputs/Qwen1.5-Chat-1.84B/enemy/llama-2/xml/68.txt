<?xml version="1.0" ?>
<game>
	<enemy>
			$id$ id
			$name$ name
			"description$ description"
			$stats$
				<hp>$hp_int_value$</hp>
				$mp>$mp_int_value$</mp>
				$atk>$atk_int_value$</atk>
				$def>$def_int_value$</def>
				$spd>$spd_int_value$</spd>
			$stats$
	<enemy>
</game>
```
New values:
- `$id`: The ID of the enemy.
- `$name`: The name of the enemy.
- `$description`: A brief description of the enemy.
- `$stats`: A nested object containing the statistics information for the enemy, including `hp`, `mp`, `atk`, `def`, and `spd`. The values are stored as string keys with corresponding integer value.

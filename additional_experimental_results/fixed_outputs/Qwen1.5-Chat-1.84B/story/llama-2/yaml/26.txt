```yaml
game:
  beginning: the beginning of the game
```

以下是Python实现的代码：

```python
def process_yaml(yaml_data):
    # Remove leading/trailing whitespace from YAML data
    yaml_data = yaml_data.strip()

    # Initialize variables to store the current token and its index
    current_token = None
    current_index = 0

    # Iterate over each character in the YAML data
    while True:
        # Check if we have reached a newline character (`\n`)
        ifyaml_data[current_index].isspace():
            # If we do, break out of the loop
            break
        # If not, check if the current token is a `>` symbol
        elif yaml_data[current_index] == '>':
            # If it is, replace the current token with `<>
            current_token = '<'
            current_index += 1
        # Otherwise, check if we have reached the end of the YAML data
        elif not yaml_data[current_index]:
            break

    # Replace all `<>` symbols in the YAML data with `>`
    new_yaml_data = yaml_data.replace('<>', '>')

    return new_yaml_data

yaml_data = '''
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
'''

new_yaml_data = process_yaml(yaml_data)
print(new_yaml_data)
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<gms:dialog>
	<gms:speaker>
		<gms:text><![CDATA[dialogue text]]></ggs:text>
	</gms:speaker>
	<gms:order>
		<gms:text><![CDATA[dialogue order]]></ggs:text>
		// only keep the output of the inner dialog text here
	</gms:order>
</gms:dialog>
```
The first example is an XML formatted Markdown code block that generates an HTML page to illustrate the concept of returning output in XML format from a Markdown code block. The XHTML-formatted code block contains a dialog text that the example uses to demonstrate the Markdown syntax. Each part of the dialog is enclosed in a `<gms:dialog>` tag, with the `text` tag containing the dialogue or narration inside the `<gms:text>` tags. The `text` tags have the `g:raw` attribute, which allows the author to output text without the inner dialog tags. The outer `<game>` tags contain the nested `<gms:dialog>` tags that wrap the `text` tags. The `order` tag is wrapped in the `<gms:text>` tag with the `g:raw` attribute set to output it as raw text without inner tags. The rest of the inner `<gms:dialog>` tags contain the nested `<gms:dialog>` tags to create the dialog structure.

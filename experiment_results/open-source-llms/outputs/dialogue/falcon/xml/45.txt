XML format:
```
<?xml version="1.0"?>
<game>\n
<dialogues>\n
<dialogue>\n
<order>dialogue order</order>\n
<speaker>\n
<text>dialogue or narration</text>\n
</speaker>\n
</dialogue>\n
<\/game>
```
For the example above, the only XML in the Markdown code block is `dialogues`. The rest of the code block, including `game`, `dialogue`, and `speaker` are Markdown tags and should be replaced with your own XML tags.

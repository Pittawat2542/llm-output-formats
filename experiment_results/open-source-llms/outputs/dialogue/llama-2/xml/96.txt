

# Example
```xml
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Sarah</speaker>
			<text>Hello, how are you?</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>John</speaker>
			<text>I am fine, thanks.</text>
		</dialogue>
	</dialogues>
</game>
```

# Notes

* The output should be formatted as an XML file.
* The dialogue order is the order in which the dialogues should be read.
* The speaker is the name of the character who is speaking.
* The text is the dialogue or narration.

# Hints

* The XML format is very simple.
* The game can be a text-based game.
* You can use any format for the dialogues.
* The dialogues can be in the form of a list.
* The dialogues can be in the form of a table.
* The dialogues can be in the form of a paragraph.
* The dialogues can be in the form of a list.
* The dialogues can be in the form of a table.
* The dialogues can be in the form of a paragraph.

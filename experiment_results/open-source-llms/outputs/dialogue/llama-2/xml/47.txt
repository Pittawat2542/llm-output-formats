

# Example
```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Alice</speaker>
			<text>Alice: I'm going to the store. Do you want anything?</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Bob</speaker>
			<text>Bob: Sure. I want a pizza.</text>
		</dialogue>
	</dialogues>
</game>
```

# Input format
```markdown
[SYS]
```

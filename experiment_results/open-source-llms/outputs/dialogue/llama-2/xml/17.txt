

## Input format

### Example
```xml
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Alex</speaker>
			<text>Alex: I'm so hungry.</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Jess</speaker>
			<text>Jess: I'm so hungry.</text>
		</dialogue>
	</dialogues>
</game>
```

### XML structure
```xml
<game>
	<dialogues>
		<dialogue>
			<order>order number</order>
			<speaker>speaker name</speaker>
			<text>dialogue or narration</text>
		</dialogue>
	</dialogues>
</game>
```

### Required fields
- `order` - the order of the dialogue
- `speaker` - the name of the speaker
- `text` - the dialogue or narration

### Optional fields
- `time` - the time when the dialogue is taking place
- `location` - the location where the dialogue is taking place

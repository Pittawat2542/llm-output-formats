

# Example
```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>dialogue order</order>
			<speaker>speaker name</speaker>
			<text>dialogue or narration</text>
		</dialogue>
	</dialogues>
</game>
```

# Usage
```python
from prompt_toolkit.output import Output
from prompt_toolkit.completion import Completion

completion = Completion()
output = Output(completion=completion)

with output.get_output_context() as ctx:
    output.push_context(ctx)
    output.run_command('INST')
```

# Command
### `INST`
Generate a dialogue.

# Arguments
* **`text`** - the text of the dialogue.

# Examples
```bash
INST "Hello, world!"
```

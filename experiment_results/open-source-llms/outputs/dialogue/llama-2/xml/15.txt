

# Usage
```
$ python3 dialogue.py
```

# Example
```
$ python3 dialogue.py

<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Alice</speaker>
			<text>Hello, Bob. How are you?</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Bob</speaker>
			<text>I'm fine, Alice. How are you?</text>
		</dialogue>
	</dialogues>
</game>
```

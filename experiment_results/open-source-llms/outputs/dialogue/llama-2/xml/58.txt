

## Input format

The input should be a Markdown document containing a dialogue. The dialogue should be in the form of a paragraph.

The dialogue should be formatted as follows:

- The dialogue should start with an opening speech tag.
- The dialogue should be indented.
- The dialogue should end with a closing speech tag.

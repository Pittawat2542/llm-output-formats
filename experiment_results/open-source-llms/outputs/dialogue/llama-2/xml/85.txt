

# Examples
### Example 1
```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Alice</speaker>
			<text>Alice: Hello, Bob. How are you?</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Bob</speaker>
			<text>Bob: I'm fine. How are you?</text>
		</dialogue>
	</dialogues>
</game>
```

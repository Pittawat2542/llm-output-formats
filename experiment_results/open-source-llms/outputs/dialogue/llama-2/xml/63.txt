

## Input format

### Example
```
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Alice</speaker>
			<text>Alice: "Hello, Bob."</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Bob</speaker>
			<text>Bob: "Hello, Alice."</text>
		</dialogue>
	</dialogues>
</game>
```

### Explanation

The input is a game with a dialogue system.

### Structure

The input is a `game` element that contains one or more `dialogues` elements.

The `dialogues` element contains zero or more `dialogue` elements.

The `dialogue` element contains a `order` attribute and a `speaker` element.

The `order` attribute is an integer that is used to sort the dialogue elements.

The `speaker` element contains a `name` attribute.

The `name` attribute is a string that identifies the speaker.

The `text` element contains a string that contains the dialogue or narration.

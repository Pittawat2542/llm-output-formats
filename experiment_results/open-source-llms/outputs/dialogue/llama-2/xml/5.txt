

# Example

```xml
<?xml version="1.0"?>
<game>
	<dialogues>
		<dialogue>
			<order>1</order>
			<speaker>Bob</speaker>
			<text>I'm Bob. I'm a nice guy.</text>
		</dialogue>
		<dialogue>
			<order>2</order>
			<speaker>Alice</speaker>
			<text>I'm Alice. I'm a nice girl.</text>
		</dialogue>
	</dialogues>
</game>
```



# Input format

```json
{
    "game": {
        "dialogues": [{
            "order": "dialogue order",
            "speaker": "speaker name",
            "text": "dialogue or narration"
        }]
    }
}
``` [/INST]

# Example

```json
{
    "game": {
        "dialogues": [{
            "order": "dialogue order",
            "speaker": "speaker name",
            "text": "dialogue or narration"
        }]
    }
}
``` [/INST]

# Hints

* Use the [JSON formatter](https://jsonformatter.curiousconcept.com/) to format the output.
* Use the [JSON formatter](https://jsonformatter.curiousconcept.com/) to format the input.
* The game's dialogues should be in the format of the example.

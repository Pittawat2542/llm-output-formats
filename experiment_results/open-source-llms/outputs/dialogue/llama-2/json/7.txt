

## Input format

```
[
  {
    "question": "question",
    "answer": "answer"
  }
]
``` [/INST]

## Example

```
[
  {
    "question": "What is your name?",
    "answer": "My name is John."
  },
  {
    "question": "How are you?",
    "answer": "I am fine."
  },
  {
    "question": "What is your favorite color?",
    "answer": "My favorite color is green."
  }
]
``` [/INST]

## Expected Output

```json
{
    "game": {
        "dialogues": [{
            "order": "dialogue order",
            "speaker": "speaker name",
            "text": "dialogue or narration"
        }]
    }
}
``` [/INST]

## Scoring

Your score is calculated by the number of correct answers.

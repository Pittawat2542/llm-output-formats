

## Input format

The input is a Markdown file with the following format:

```
<</SYS>>

[INST]

# Dialogue

[TEXT]

[/INST]
```

The dialogue is a sequence of sentences that are spoken by one or more characters.

# Narration

Narration is a sequence of sentences that are spoken by a single character.

# Markdown

Markdown is a lightweight markup language that allows you to format text.

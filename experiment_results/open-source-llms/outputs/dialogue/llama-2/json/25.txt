

### Input

Input is a list of questions in JSON format.

# Example

```json
[
    {
        "question": "What is the best way to get to the city center from the airport?"
    },
    {
        "question": "What is the best way to get to the city center from the train station?"
    },
    {
        "question": "What is the best way to get to the city center from the bus station?"
    }
]
```

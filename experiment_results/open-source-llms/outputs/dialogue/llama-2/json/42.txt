

## Example

```json
{
    "game": {
        "dialogues": [{
            "order": "dialogue order",
            "speaker": "speaker name",
            "text": "dialogue or narration"
        }]
    }
}
```

### Required

- **game**: A JSON object containing the dialogues of the game.

### Optional

- **speaker**: The name of the speaker.
- **text**: The text of the dialogue.
- **order**: The order of the dialogue.

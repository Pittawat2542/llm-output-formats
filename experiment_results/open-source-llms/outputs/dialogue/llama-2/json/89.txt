

## Input format

### Example

```json
{
    "game": {
        "dialogues": [{
            "order": "dialogue order",
            "speaker": "speaker name",
            "text": "dialogue or narration"
        }]
    }
}
```

### Required fields

* **game** - the game object
* **dialogues** - an array of dialogues

### Optional fields

* **order** - the order of the dialogue
* **speaker** - the speaker of the dialogue
* **text** - the dialogue or narration

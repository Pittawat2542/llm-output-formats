

### Example
```yaml
game:
  dialogues:
  - order: 1
    speaker: player
    text: "I'm hungry."
  - order: 2
    speaker: player
    text: "I'm thirsty."
  - order: 3
    speaker: player
    text: "I'm tired."
  - order: 4
    speaker: player
    text: "I'm bored."
  - order: 5
    speaker: player
    text: "I'm sad."
  - order: 6
    speaker: player
    text: "I'm happy."
```

### Specification

#### Game

| Name | Description | Required |
| ------------- | ------------- | ------------- |
| game | The game to use | Yes |
| dialogues | Dialogues to use | Yes |

#### Dialogues

| Name | Description | Required |
| ------------- | ------------- | ------------- |
| order | Order of dialogue | Yes |
| speaker | Speaker of dialogue | Yes |
| text | Dialogue | Yes |

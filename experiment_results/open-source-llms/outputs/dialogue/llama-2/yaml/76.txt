

## Input

### Example

```yaml
game:
  dialogues:
  - order: dialogue order
    speaker: speaker name
    text: dialogue or narration
```

### Rules

- The input should be a single YAML document.
- The document should be indented by 2 spaces.
- The document should be in a single line.
- The document should not contain any leading or trailing whitespace.
- The document should not contain any comments.
- The document should not contain any leading or trailing comments.
- The document should not contain any leading or trailing blank lines.

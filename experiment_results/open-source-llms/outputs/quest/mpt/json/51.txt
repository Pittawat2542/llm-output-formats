## Game
Game related information.

# Properties
- id: Global game identifier, required input.
- title: Title of the game, required input.
- objective: Objective of the quest, required input.
- description: Input description, required input.
- reward: Reward of the quest, required input.
- quest_giver: Name of the giver who provides this quest, required input.
- tasks: The list of tasks in the quest.
# Fields
- order: Number of the task to perform first in the list.
- objective & description: Task description, required input. Location is optional.
- location: Location of the task, required input.

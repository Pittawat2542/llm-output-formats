```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

The game object includes basic information about the quest, such as the identifier, title, objective, description, and reward of the quest.

The tasks array contains all the tasks of the quest. Each element of the array represents an individual task and includes information on the order of the task, its objective, description, and location.

```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "1",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        },
        {
            "order": "2",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

* game - an object containing data for the quest as a whole
    * id - quest id
    * title - quest title
    * objective - quest objective
    * description - quest description
    * reward - quest reward
    * quest_giver - quest giver
    * tasks - an array of objects containing data about individual tasks

        * order - number of the task
        * objective - task objective
        * description - task description
        * location - task location



# Input format

## Game

- **id**: a unique identifier for the game
- **title**: the game's name
- **objective**: the game's objective
- **description**: the game's description
- **reward**: the game's reward
- **quest_giver**: the game's quest giver

## Quest

- **id**: a unique identifier for the quest
- **game**: the game that the quest belongs to
- **title**: the quest's name
- **objective**: the quest's objective
- **description**: the quest's description
- **reward**: the quest's reward
- **quest_giver**: the quest's quest giver

## Task

- **order**: the task's order
- **objective**: the task's objective
- **description**: the task's description
- **location**: the task's location

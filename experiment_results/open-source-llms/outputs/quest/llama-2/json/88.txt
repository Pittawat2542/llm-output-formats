

## Input format

### Game

#### Game ID

The game ID is a unique identifier for the game.

#### Game Title

The game title is the name of the game.

#### Game Objective

The game objective is the objective of the game.

#### Game Description

The game description is a description of the game.

#### Game Reward

The game reward is the reward for completing the game.

#### Game Quest Giver

The game quest giver is the person who gives the quest.

### Quest

#### Quest ID

The quest ID is a unique identifier for the quest.

#### Quest Title

The quest title is the name of the quest.

#### Quest Objective

The quest objective is the objective of the quest.

#### Quest Description

The quest description is a description of the quest.

#### Quest Reward

The quest reward is the reward for completing the quest.

#### Quest Giver

The quest giver is the person who gives the quest.

#### Quest Tasks

The quest tasks are the tasks that need to be completed in order to complete the quest.

### Task

#### Task Order

The task order is the order in which the task needs to be completed.

#### Task Objective

The task objective is the objective of the task.

#### Task Description

The task description is a description of the task.

#### Task Location

The task location is the location where the task needs to be completed.



## Inputs

### Game

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| id | string | Yes | Game ID |
| title | string | Yes | Quest title |
| objective | string | Yes | Quest objective |
| description | string | Yes | Quest description |
| reward | string | Yes | Quest reward |
| quest_giver | string | Yes | Quest giver |
| tasks | array | Yes | Quest tasks |

### Task

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| order | integer | Yes | Task order |
| objective | string | Yes | Task objective |
| description | string | Yes | Task description |
| location | string | Yes | Task location |

### Example

```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

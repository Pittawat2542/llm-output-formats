

## Input format

### Game

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| id | string | Yes | Game ID |
| title | string | Yes | Game title |
| objective | string | Yes | Game objective |
| description | string | Yes | Game description |
| reward | string | Yes | Game reward |
| quest_giver | string | Yes | Game quest giver |
| tasks | array | Yes | Tasks |

### Task

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| order | number | Yes | Task order |
| objective | string | Yes | Task objective |
| description | string | Yes | Task description |
| location | string | Yes | Task location |

### Example

```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

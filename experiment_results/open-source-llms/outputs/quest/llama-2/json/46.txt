

## Input format
### Parameters
* **quest_id** (required) - The ID of the quest.

### Example
```json
{
    "quest_id": "id"
}
``` [/INST]

### Output format
```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
``` [/INST]

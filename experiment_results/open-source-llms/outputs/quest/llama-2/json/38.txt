

# Input format

```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

# Input validation

* `game` must be an object
* `game.id` must be a string
* `game.title` must be a string
* `game.objective` must be a string
* `game.description` must be a string
* `game.reward` must be a string
* `game.quest_giver` must be a string
* `game.tasks` must be an array of objects
* `game.tasks[].order` must be a string
* `game.tasks[].objective` must be a string
* `game.tasks[].description` must be a string
* `game.tasks[].location` must be a string

# Output validation

* `game` must be an object
* `game.id` must be a string
* `game.title` must be a string
* `game.objective` must be a string
* `game.description` must be a string

# Errors

* If `game` is not an object, an error will be returned
* If `game.id` is not a string, an error will be returned
* If `game.title` is not a string, an error will be returned
* If `game.objective` is not a string, an error will be returned
* If `game.description` is not a string, an error will be returned
* If `game.reward` is not a string, an error will be returned
* If `game.quest_giver` is not a string, an error will be returned
* If `game.tasks` is not an array of objects, an error will be returned
* If `game.tasks[].order` is not a string, an error will be returned
* If `game.tasks[].objective` is not a string, an error will be returned
* If `game.tasks[].description` is not a string, an error will be returned
* If `game.tasks[].location` is not a string, an error will be returned

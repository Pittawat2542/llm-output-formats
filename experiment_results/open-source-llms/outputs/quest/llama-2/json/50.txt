

## Input format

### Command line
```
python3 main.py [--help] [--version] [--inst] [--sys] [--input] [--output] [--game] [--id] [--title] [--objective] [--description] [--reward] [--quest_giver] [--tasks] [--order] [--objective] [--description] [--location] [--output] [--help] [--version] [--inst] [--sys] [--input] [--output]
```

### JSON
```
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

### Example
```
python3 main.py --inst
```

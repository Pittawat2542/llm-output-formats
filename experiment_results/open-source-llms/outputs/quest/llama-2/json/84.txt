### Inputs

#### Game

| Name | Type | Description | Required |
| --- | --- | --- | --- |
| id | String | Game ID | Yes |
| title | String | Game title | Yes |
| objective | String | Game objective | Yes |
| description | String | Game description | Yes |
| reward | String | Game reward | Yes |
| quest_giver | String | Game quest giver | Yes |
| tasks | Array | Game tasks | Yes |

#### Task

| Name | Type | Description | Required |
| --- | --- | --- | --- |
| order | Number | Task order | Yes |
| objective | String | Task objective | Yes |
| description | String | Task description | Yes |
| location | String | Task location | Yes |

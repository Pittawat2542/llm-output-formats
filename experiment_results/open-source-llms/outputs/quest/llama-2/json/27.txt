

# Input format

### Game information
```json
{
    "id": "id",
    "title": "quest title",
    "objective": "quest objective",
    "description": "quest description",
    "reward": "quest reward",
    "quest_giver": "quest giver",
    "tasks": [{
        "order": "task order",
        "objective": "task objective",
        "description": "task description",
        "location": "task location"
    }]
}
``` [/INST]

### Task information
```json
{
    "order": "task order",
    "objective": "task objective",
    "description": "task description",
    "location": "task location"
}
``` [/INST]

# Usage
```bash
python3 quests.py
```



## Input format

### JSON
```json
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "task objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

### Markdown
```md
---
title: Quest
---

### Quest Information

| Field | Description |
| --- | --- |
| game.id | ID of the game |
| game.title | Title of the game |
| game.objective | Quest objective |
| game.description | Quest description |
| game.reward | Quest reward |
| game.quest_giver | Quest giver |
| game.tasks | Tasks |

### Tasks
| Field | Description |
| --- | --- |
| order | Task order |
| objective | Task objective |
| description | Task description |
| location | Task location |
```



## Input format

### Example

```xml
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

### Required fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| id | string | yes | Unique identifier for the quest. |
| title | string | yes | Quest title. |
| objective | string | yes | Quest objective. |
| description | string | yes | Quest description. |
| reward | string | yes | Quest reward. |
| quest-giver | string | yes | Quest giver. |
| tasks | array | yes | Array of tasks. |

### Optional fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| location | string | no | Quest location. |

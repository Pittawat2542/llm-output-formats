

## Input format
```xml
<?xml version="1.0"?>
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
``` [/INST]

## Example
```xml
<?xml version="1.0"?>
<game>
	<id>1</id>
	<title>The quest</title>
	<objective>Find the treasure</objective>
	<description>Find the treasure in the dungeon</description>
	<reward>A magic wand</reward>
	<quest-giver>The king</quest-giver>
	<tasks>
		<task>
			<order>1</order>
			<objective>Find the treasure</objective>
			<description>Find the treasure in the dungeon</description>
			<location>The dungeon</location>
		</task>
		<task>
			<order>2</order>
			<objective>Find the treasure</objective>
			<description>Find the treasure in the dungeon</description>
			<location>The dungeon</location>
		</task>
	</tasks>
</game>
``` [/INST]

## Notes
- The task order should be in ascending order.
- The task description should be a valid sentence.
- The task location should be a valid sentence.
- The quest reward should be a valid sentence.
- The quest giver should be a valid sentence.

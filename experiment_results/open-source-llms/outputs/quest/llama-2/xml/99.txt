

### Input format

#### Example

```xml
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

### Rules

#### Id

The `id` element must contain a unique identifier for the quest.

#### Title

The `title` element must contain the name of the quest.

#### Objective

The `objective` element must contain the objective of the quest.

#### Description

The `description` element must contain a description of the quest.

#### Reward

The `reward` element must contain the reward of the quest.

#### Quest-giver

The `quest-giver` element must contain the name of the quest giver.

#### Tasks

The `tasks` element must contain a list of tasks.

#### Order

The `order` element must contain the order of the task.

#### Objective

The `objective` element must contain the objective of the task.

#### Description

The `description` element must contain a description of the task.

#### Location

The `location` element must contain the location of the task.

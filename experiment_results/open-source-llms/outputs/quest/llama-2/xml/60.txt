

# Input format
```xml
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
``` [/INST]

# Input validation

* Quest ID must be a string.
* Quest title must be a string.
* Quest objective must be a string.
* Quest description must be a string.
* Quest reward must be a string.
* Quest giver must be a string.
* Tasks must be an array.
* Tasks must be an array of objects.
* Task order must be a string.
* Task objective must be a string.
* Task description must be a string.
* Task location must be a string.

# Output validation

* XML must be valid.

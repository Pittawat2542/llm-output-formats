

## Input

### Parameters

| Name | Description | Type | Required | Default |
|------|-------------|------|----------|---------|
| id | Quest ID | int | Yes | 0 |
| title | Quest title | string | Yes |  |
| objective | Quest objective | string | Yes |  |
| description | Quest description | string | Yes |  |
| reward | Quest reward | string | Yes |  |
| quest-giver | Quest giver | string | Yes |  |
| tasks | Tasks | [Task](#task) | Yes |  |

### Task

| Name | Description | Type | Required | Default |
|------|-------------|------|----------|---------|
| order | Task order | int | Yes | 0 |
| objective | Task objective | string | Yes |  |
| description | Task description | string | Yes |  |
| location | Task location | string | Yes |  |

### Example

```xml
<game>
	<id>1</id>
	<title>The quest title</title>
	<objective>The quest objective</objective>
	<description>The quest description</description>
	<reward>The quest reward</reward>
	<quest-giver>The quest giver</quest-giver>
	<tasks>
		<task>
			<order>1</order>
			<objective>The task objective</objective>
			<description>The task description</description>
			<location>The task location</location>
		</task>
	</tasks>
</game>
```

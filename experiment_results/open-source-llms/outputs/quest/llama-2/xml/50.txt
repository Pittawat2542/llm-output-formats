

## Input format

# Input format

### Questions

Questions should be in the format:

```
[QUESTION]
Question
[/QUESTION]
```

### Answers

Answers should be in the format:

```
[ANSWER]
Answer
[/ANSWER]
```

### Examples

```
[QUESTION]
What is the capital of the United States?
[/QUESTION]
[ANSWER]
Washington, D.C.
[/ANSWER]
```

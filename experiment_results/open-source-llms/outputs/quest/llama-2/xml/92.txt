

## Input

### Parameters

| Parameter | Type | Description |
| :--- | :--- | :--- |
| id | string | The unique identifier of the quest. |
| title | string | The title of the quest. |
| objective | string | The objective of the quest. |
| description | string | The description of the quest. |
| reward | string | The reward of the quest. |
| quest-giver | string | The quest giver. |
| tasks | array | The tasks of the quest. |

### Tasks

| Parameter | Type | Description |
| :--- | :--- | :--- |
| order | integer | The order of the task. |
| objective | string | The objective of the task. |
| description | string | The description of the task. |
| location | string | The location of the task. |

### Example

```json
{
	"id": "id",
	"title": "quest title",
	"objective": "quest objective",
	"description": "quest description",
	"reward": "quest reward",
	"quest-giver": "quest giver",
	"tasks": [
		{
			"order": 1,
			"objective": "task objective",
			"description": "task description",
			"location": "task location"
		}
	]
}
```

## Output

### Example

```xml
<?xml version="1.0"?>
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

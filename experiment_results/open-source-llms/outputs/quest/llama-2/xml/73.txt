

# Input format

### Quest information

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| id | string | yes | The quest ID. |
| title | string | yes | The quest title. |
| objective | string | yes | The quest objective. |
| description | string | yes | The quest description. |
| reward | string | yes | The quest reward. |
| quest-giver | string | yes | The quest giver. |
| tasks | array | yes | The quest tasks. |

### Quest tasks

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| order | integer | yes | The task order. |
| objective | string | yes | The task objective. |
| description | string | yes | The task description. |
| location | string | yes | The task location. |

### Quest information example

```json
{
	"id": "1234",
	"title": "The quest title",
	"objective": "The quest objective",
	"description": "The quest description",
	"reward": "The quest reward",
	"quest-giver": "The quest giver",
	"tasks": [
		{
			"order": 1,
			"objective": "The task objective",
			"description": "The task description",
			"location": "The task location"
		}
	]
}
```

### Task information example

```json
{
	"order": 1,
	"objective": "The task objective",
	"description": "The task description",
	"location": "The task location"
}
```

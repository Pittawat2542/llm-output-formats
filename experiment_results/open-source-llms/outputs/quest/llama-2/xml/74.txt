

## Input

### Required

| Name | Type | Description |
| ---- | ---- | ----------- |
| id | string | Unique ID of the quest. |
| title | string | Quest title. |
| objective | string | Quest objective. |
| description | string | Quest description. |
| reward | string | Quest reward. |
| quest-giver | string | Quest giver. |

### Optional

| Name | Type | Description |
| ---- | ---- | ----------- |
| tasks | array | Tasks of the quest. |

## Example

### Input

```json
{
	"id": "id",
	"title": "quest title",
	"objective": "quest objective",
	"description": "quest description",
	"reward": "quest reward",
	"quest-giver": "quest giver"
}
```

### Output

```xml
<?xml version="1.0"?>
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

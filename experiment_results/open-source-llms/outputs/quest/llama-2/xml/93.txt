

# Input format
```json
{
	"id": "id",
	"title": "quest title",
	"objective": "quest objective",
	"description": "quest description",
	"reward": "quest reward",
	"quest-giver": "quest giver",
	"tasks": [
		{
			"order": "task order",
			"objective": "task objective",
			"description": "task description",
			"location": "task location"
		}
	]
}
``` [/INST]

# Example
```json
{
	"id": "12345",
	"title": "The Wizard's Quest",
	"objective": "Find the Wizard's magic wand",
	"description": "You will need to find the Wizard's magic wand to complete your quest",
	"reward": "A magic wand",
	"quest-giver": "The Wizard",
	"tasks": [
		{
			"order": "1",
			"objective": "Find the Wizard's magic wand",
			"description": "You will need to find the Wizard's magic wand to complete your quest",
			"location": "The Wizard's house"
		}
	]
}
``` [/INST]

# Tasks
## Input
```json
{
	"order": "1",
	"objective": "Find the Wizard's magic wand",
	"description": "You will need to find the Wizard's magic wand to complete your quest",
	"location": "The Wizard's house"
}
``` [/INST]

## Output
```xml
<task order="1">
	<objective>Find the Wizard's magic wand</objective>
	<description>You will need to find the Wizard's magic wand to complete your quest</description>
	<location>The Wizard's house</location>
</task>
``` [/INST]

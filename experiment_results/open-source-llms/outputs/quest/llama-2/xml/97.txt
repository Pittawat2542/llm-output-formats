

## Input format

The input is a text file containing a list of questions. Each question is in the format:

```
Question:
Answer:
```

The first line of the text file contains the question. The second line contains the answer.

The text file should be named "questions.txt".

## Example

```
Question:
What is the name of the game?

Answer:
The name of the game is "Quest".
```

## Sample output

```xml
<?xml version="1.0"?>
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

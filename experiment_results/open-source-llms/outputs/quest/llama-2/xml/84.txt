

## Examples
```xml
<?xml version="1.0"?>
<game>
	<id>1234567890</id>
	<title>Hero's Quest</title>
	<objective>Save the princess from the evil wizard</objective>
	<description>You are a hero, and you must save the princess from the evil wizard.</description>
	<reward>100 gold coins</reward>
	<quest-giver>The king</quest-giver>
	<tasks>
		<task>
			<order>1</order>
			<objective>Find the princess</objective>
			<description>You must find the princess in the dungeon.</description>
			<location>The dungeon</location>
		</task>
		<task>
			<order>2</order>
			<objective>Defeat the evil wizard</objective>
			<description>You must defeat the evil wizard to save the princess.</description>
			<location>The evil wizard's castle</location>
		</task>
	</tasks>
</game>
```

## Inputs

### id

The game id.

### title

The game title.

### objective

The game objective.

### description

The game description.

### reward

The game reward.

### quest-giver

The game quest giver.

### tasks

The game tasks.

### task

The game task.

### order

The game task order.

### objective

The game task objective.

### description

The game task description.

### location

The game task location.

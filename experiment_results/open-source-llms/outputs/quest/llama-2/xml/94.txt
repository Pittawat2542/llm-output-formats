

# Input format

```
<game>
	<id>id</id>
	<title>quest title</title>
	<objective>quest objective</objective>
	<description>quest description</description>
	<reward>quest reward</reward>
	<quest-giver>quest giver</quest-giver>
	<tasks>
		<task>
			<order>task order</order>
			<objective>task objective</objective>
			<description>task description</description>
			<location>task location</location>
		</task>
	</tasks>
</game>
```

# Rules

*   All fields in the input file must be present in the output file.
*   All fields in the input file must have the correct format.
*   All fields in the input file must have the correct content.
*   All fields in the input file must be in the correct order.
*   All fields in the input file must be correctly aligned.
*   All fields in the input file must be correctly indented.



# Input format
```
<game id="id" title="quest title" objective="quest objective" description="quest description" reward="quest reward" quest-giver="quest giver" tasks="
	<task order="task order" objective="task objective" description="task description" location="task location">
	</task>
" />
``` [/INST]

# Usage
```
python3 quest.py
```

# Examples
```
python3 quest.py
```

# Output
```
<?xml version="1.0"?>
<game id="1" title="quest title" objective="quest objective" description="quest description" reward="quest reward" quest-giver="quest giver" tasks="
	<task order="1" objective="task objective" description="task description" location="task location">
	</task>
" />

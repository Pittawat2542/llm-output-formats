

## Input format

### Quest ID

```yaml
id: quest_id
```

### Quest information

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

### Quest Giver

```yaml
quest_giver: quest_giver
```

### Quest Reward

```yaml
reward: quest_reward
```

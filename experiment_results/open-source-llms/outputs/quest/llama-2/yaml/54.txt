

## Input format
```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

## Example
```yaml
game:
  description: "The quest for the Holy Grail."
  id: 1
  objective: "Find the Holy Grail."
  quest_giver: "King Arthur."
  reward: "A golden chalice."
  tasks:
  - description: "Find the Holy Grail."
    location: "Camelot."
    objective: "Find the Holy Grail."
    order: 1
  - description: "Find the Holy Grail."
    location: "Camelot."
    objective: "Find the Holy Grail."
    order: 2
  - description: "Find the Holy Grail."
    location: "Camelot."
    objective: "Find the Holy Grail."
    order: 3
  title: "The quest for the Holy Grail."
```

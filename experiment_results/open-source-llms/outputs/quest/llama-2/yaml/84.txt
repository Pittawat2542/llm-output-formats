

## Usage
```
$ python3 quest.py
```

### Input
```
$ python3 quest.py -h
usage: quest.py [-h] -g GAME_ID -q QUEST_ID

Generate a quest information. Return output in YAML format and only the YAML in the Markdown
code block.

optional arguments:
  -h, --help            show this help message and exit
  -g GAME_ID, --game_id GAME_ID
                        game id
  -q QUEST_ID, --quest_id QUEST_ID
                        quest id
```

### Output
```
$ python3 quest.py -g 24 -q 24
```

### Example
```
$ python3 quest.py -g 24 -q 24

game:
  description: quest description
  id: 24
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

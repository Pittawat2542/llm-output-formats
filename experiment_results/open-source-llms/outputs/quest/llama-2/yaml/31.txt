

## Input format

### Example
```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

### Required fields

#### game

* **description**: quest description
* **id**: quest id
* **objective**: quest objective
* **quest_giver**: quest giver
* **reward**: quest reward
* **tasks**: list of tasks
* **title**: quest title

### Optional fields

#### game.tasks

* **description**: task description
* **location**: task location
* **objective**: task objective
* **order**: task order



# Input format
```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
``` [/INST]

# Example
```yaml
game:
  description: A quest to get a quest reward
  id: 1
  objective: Get quest reward
  quest_giver: Someone
  reward: A quest reward
  tasks:
  - description: Get a quest reward
    location: Somewhere
    objective: Get quest reward
    order: 1
  - description: Get a quest reward
    location: Somewhere
    objective: Get quest reward
    order: 2
  - description: Get a quest reward
    location: Somewhere
    objective: Get quest reward
    order: 3
  title: A quest to get a quest reward
``` [/INST]

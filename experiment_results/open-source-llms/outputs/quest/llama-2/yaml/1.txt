

# Example
```yaml
game:
  description: A simple quest
  id: 1
  objective: Find the lost key.
  quest_giver: The king
  reward: A golden key
  tasks:
  - description: Find the key.
    location: The castle
    objective: Find the key.
    order: 1
  - description: Find the key.
    location: The castle
    objective: Find the key.
    order: 2
  title: Find the lost key.
```

# Input format

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

# Tasks

```yaml
tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
```

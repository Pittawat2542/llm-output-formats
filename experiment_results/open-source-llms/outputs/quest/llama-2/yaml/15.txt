

# Input format

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

# Example

```yaml
game:
  description: A quest to find the treasure
  id: 1
  objective: Find the treasure
  quest_giver: The King
  reward: 1000 gold
  tasks:
  - description: Find the treasure
    location: The king's castle
    objective: Find the treasure
    order: 1
  - description: Find the treasure
    location: The king's castle
    objective: Find the treasure
    order: 2
  - description: Find the treasure
    location: The king's castle
    objective: Find the treasure
    order: 3
  - description: Find the treasure
    location: The king's castle
    objective: Find the treasure
    order: 4
  title: A quest to find the treasure
```

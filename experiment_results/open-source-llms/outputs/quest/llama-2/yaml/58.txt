

## Input format

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

## Examples

```yaml
game:
  description: "I need to find my lost cat"
  id: 1
  objective: "Find my lost cat"
  quest_giver: "Me"
  reward: "A cookie"
  tasks:
  - description: "Find the cat"
    location: "My house"
    objective: "Find the cat"
    order: 1
  - description: "Give the cat some food"
    location: "My house"
    objective: "Give the cat some food"
    order: 2
  - description: "Give the cat a bath"
    location: "My house"
    objective: "Give the cat a bath"
    order: 3
  - description: "Give the cat a toy"
    location: "My house"
    objective: "Give the cat a toy"
    order: 4
  title: "Find my lost cat"
```

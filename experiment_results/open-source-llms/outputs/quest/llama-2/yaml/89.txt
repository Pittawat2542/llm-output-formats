

## Input format

### Required

- `game_id` - the id of the game.
- `quest_id` - the id of the quest.

### Optional

- `quest_giver` - the name of the quest giver.
- `quest_title` - the title of the quest.
- `quest_description` - the description of the quest.
- `quest_objective` - the objective of the quest.
- `quest_reward` - the reward for the quest.
- `tasks` - an array of tasks.

### Tasks

- `task_order` - the order of the task.
- `task_location` - the location of the task.
- `task_description` - the description of the task.
- `task_objective` - the objective of the task.

## Example

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location
    objective: task objective
    order: task order
  title: quest title
```

## Example output

```yaml
game:
  description: quest description
  id: id
  objective: quest objective
  quest_giver: quest giver
  reward: quest reward
  tasks:
  - description: task description
    location: task location

```

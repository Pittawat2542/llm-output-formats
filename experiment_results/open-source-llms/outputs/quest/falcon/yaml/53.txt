# Markdown code block
| Description | Required Inputs (Defaults in `YAML.strict_list`) | YAML |
|---------------------------------------------------------------------------|-------------------------|--------|
| Game description | `description: game description` | `--` |
| Quest objectives | `objective: @title` | `--` |
| Quest giver / companions | `giver: @title` | `--` |
| Location (quest description) | `location: location` | `--` |
| Objectives (quest description) | `objective: @title` | `--` |
| X marks the quest giver locations | `x: location_name`: | `--` |
| X marks the order in which tasks need to be completed | `tasks: @title` | `--` |
| Title of quest | `title: @title` | `--` |

| Quest Description | Description in Markdowns | @title |
|----------:----------:--------------------------------------:|
| Description | description in markdown             |
| ^: YAML markdown code, followed by @title. |
^    : Description of quest, including @title. |
@title: Title of quest in markdowns |
| @title: @title (if needed), followed by ":" |
<title>: Title of quest |
=: Equals markdown format marker. |
```

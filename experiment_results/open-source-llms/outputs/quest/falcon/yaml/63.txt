# Markdown code block
You can create a markdown code block by adding the following lines:
```
(name)

# Start of game information
* Game Start *
- (id)
- (game id)

# Start of quest information
- (name)
# Quest description
- (description)
    Tasks
    1. (tasks description)
- (location)
    Location (X, Y)
- (objective description)
    Objective is (text)
- (reward)
    Reward is (text)

- (tasks description)
    - (description)
    tasks description (order)
    1. (task #1 description)
    2. (task #2 description)
   ...
```
Replace the text in the YAML code block with the game and query string.

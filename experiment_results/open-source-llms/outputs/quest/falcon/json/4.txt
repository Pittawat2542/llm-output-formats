To return only the JSON content of a Markdown code block, you can use the following code block:

```markdown
{
    "game": {
        "id": "id",
        "title": "quest title",
        "objective": "quest objective",
        "description": "quest description",
        "reward": "quest reward",
        "quest_giver": "quest giver",
        "tasks": [{
            "order": "task order",
            "objective": "quest objective",
            "description": "task description",
            "location": "task location"
        }]
    }
}
```

Replace "game" in the code block with the name of your own key and "id", "title", "objective", "description", "reward", "quest_giver", and "tasks" with the respective keys in your JSON code block.

Quest Information
============================
- `game` key indicates it's a quest object
- `game.id`: Quest ID (required)
- `game.title`: Quest Title (required)
- `game.objective`: Quest Objective (required)
- `game.property.property_1`: Quest Property #1
- `game.property.property_2`: Quest Property #2
- `game.property.property_3`: Quest Property #3
- `game.quest_giver`: Quest Giver (required)
- `game.tasks`: An array of tasks in JSON format (required)
- `game"`: Object root of the quest information (required)
- `game`": Quest JSON in "JSON.stringify()" format (required)
- `game.objective`: Quest Objective JSON in "JSON.stringify()" format (required)
- `game.property.property_1`: Quest Property #1 JSON in "JSON.stringify()" format (required)
- `game.property.property_2`: Quest Property #2 JSON in "JSON.stringify()" format (required)
- `game.property.property_3`: Quest Property #3 JSON in "JSON.stringify()" format (required)

Markdown code block:
```markdown
Game:
  Character:
    age (years): exact age or description
```
```yaml
game:
  character:
    background (story): description
    first_name (string): first name
    id (string): id
    last_name (string): last name
    physical_appearance (dict):
    - eye_color (string): eye color
      hair_color (string): hair color
      height (int or float): height in meters
      weight (float): weight in grams
```

Output format:

```yaml
game:
  character:
    age: 30
    background: 
      story: The character was born into a family of powerful wizards.
      id: James Sirius Potter
```

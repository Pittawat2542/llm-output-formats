```
game
age      : Age or description of the character
background      : Brief description of the character's background
first_name      : First name of the character
id         : Character identification code
last_name     : Last name of the character
physical_appearance:
    - eye_color     : Eye color
    - hair_color    : Hair color
    - height        : Height in feet or inches
    - weight        : Weight in pounds
place_of_birth     : Place and date of birth
```
```
game.character
age        : The exact age of the character or a description of their age
background    : A brief description of the character's background
first_name    : The first name of the character
id         : The character identification code
last_name    : The last name of the character
physical_appearance  : Physical description of the character in YAML format
```

# Markdown Code Block
### Character Profile

- Game: character
- Character: 
  - id: id
  - first_name: first name
  - last_name: last name
  - species: specific species (ex. Earth-like species)
  - age: exact age (or description)
- Role: Description of character's role
- Background: Description of character's background story
- Place of Birth: Description of character's place of birth
- Physical Appearnce: 
    - eye_color: Eye color (e.g. brown, green, etc.)
    - hair_color: Hair color (ex. blonde, black, etc.)
    - height: Height in feet
    - weight: Weight in pounds



### Example
```yaml
game:
  character:
    age: 18
    background: The character has been in a long-term relationship with a person of the same gender.
    first_name: John
    id: 1234567890
    last_name: Doe
    physical_appearance:
    - eye_color: green
      hair_color: blond
      height: 180 cm
      weight: 80 kg
    place_of_birth: Earth
    role: the character is a police officer
    species: human
```

### Explanation

The output format is a YAML file. The game key is the name of the game, and the character key is the name of the character. The character key has the following keys:

- age: exact age or description
- background: background story
- first_name: first name
- id: id
- last_name: last name
- physical_appearance:
  - eye_color: eye color
  - hair_color: hair color
  - height: height in float value
  - weight: weight in float value
- place_of_birth: location
- role: role of the character
- species: species

### Input format

You can use Markdown or a single-line string.

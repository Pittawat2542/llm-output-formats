

# Input format
```yaml
game:
  character:
    age: exact age or description
    background: background story
    first_name: first name
    id: id
    last_name: last name
    physical_appearance:
    - eye_color: eye color
      hair_color: hair color
      height: height in float value
      weight: weight in float value
    place_of_birth: location
    role: role of the character
    species: species
``` [/INST]

# Example
```yaml
game:
  character:
    age: 25
    background: 
    - I am a very nice person
    first_name: 
    id: 1
    last_name: 
    physical_appearance:
    - eye_color: 
      hair_color: 
      height: 
      weight: 
    place_of_birth: 
    role: 
    species: 
``` [/INST]

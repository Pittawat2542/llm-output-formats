

# Input format
```yaml
game:
  character:
    age: exact age or description
    background: background story
    first_name: first name
    id: id
    last_name: last name
    physical_appearance:
    - eye_color: eye color
      hair_color: hair color
      height: height in float value
      weight: weight in float value
    place_of_birth: location
    role: role of the character
    species: species
``` [/INST]

# Sample
```yaml
game:
  character:
    age: 18
    background: The character was born in the city of Chicago.
    first_name: John
    id: 1
    last_name: Doe
    physical_appearance:
    - eye_color: blue
      hair_color: brown
      height: 180
      weight: 75
    place_of_birth: Chicago
    role: Student
    species: Human
``` [/INST]

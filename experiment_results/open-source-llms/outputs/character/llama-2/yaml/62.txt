

### Example

```yaml
game:
  character:
    age: 28
    background: "Grew up in a small town with her parents and two siblings."
    first_name: "Teresa"
    id: "101"
    last_name: "Hill"
    physical_appearance:
    - eye_color: blue
      hair_color: brown
      height: 5'5"
      weight: 115 lbs
    place_of_birth: "Lancaster, PA"
    role: "Athlete"
    species: "Human"
```

### Notes

* The `game` key is required.
* The `character` key is required.
* The `character.id` key is required.
* The `character.first_name` key is required.
* The `character.last_name` key is required.
* The `character.physical_appearance` key is required.
* The `character.physical_appearance.eye_color` key is required.
* The `character.physical_appearance.hair_color` key is required.
* The `character.physical_appearance.height` key is required.
* The `character.physical_appearance.weight` key is required.
* The `character.place_of_birth` key is required.
* The `character.role` key is required.
* The `character.species` key is required.

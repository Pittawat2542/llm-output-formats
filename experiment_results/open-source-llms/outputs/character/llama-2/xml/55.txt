

## Input format

You can provide the information in the following formats:

- A text file with one character per line.
- A Markdown file with one character per line.
- A Markdown file with one character per paragraph.
- A Markdown file with one character per Markdown heading.
- A Markdown file with one character per Markdown heading and paragraph.

## Examples

### Text file

```
id: 1
first-name: John
last-name: Doe
species: Human
age: 25
role: Engineer
background: John Doe was born on Earth in the 22nd century. He is a brilliant engineer and has a degree in engineering from the University of Earth.
place-of-birth: Earth
physical-appearance:
	eye-color: Blue
	hair-color: Brown
	height: 1.8 m
	weight: 80 kg
```

```json
            "species": "species",

            "role": "role of the character",
            "background": "background story",
            "place_of_birth": "location",
            "physical_appearance": [{
                "eye_color": "eye color",
                "hair_color": "hair color",
                "height": "height in float value",
                "weight": "weight in float value"
            }]
        }
    }
}
```

# Prompt
```json
{
    "game": {
        "character": {
            "id": "id",
            "first_name": "first name",
            "last_name": "last name",
            "species": "species",
            "age": "exact age or description",
            "role": "role of the character",
            "background": "background story",
            "place_of_birth": "location",
            "physical_appearance": [{
                "eye_color": "eye color",
                "hair_color": "hair color",
                "height": "height in float value",
                "weight": "weight in float value"
            }]
        }
    }
}
```

# Constraints
- **game.character.id** - Must be a string between 1 and 30 characters.
- **game.character.first_name** - Must be a string between 1 and 30 characters.
- **game.character.last_name** - Must be a string between 1 and 30 characters.
- **game.character.species** - Must be a string between 1 and 30 characters.
- **game.character.age** - Must be a string between 1 and 30 characters.
- **game.character.role** - Must be a string between 1 and 30 characters.
- **game.character.background** - Must be a string between 1 and 30 characters.
- **game.character.place_of_birth** - Must be a string between 1 and 30 characters.
- **game.character.physical_appearance** - Must be an array of objects.
  - **game.character.physical_appearance.eye_color** - Must be a string between 1 and 30 characters.
  - **game.character.physical_appearance.hair_color** - Must be a string between 1 and 30 characters.
  - **game.character.physical_appearance.height** - Must be a string between 1 and 30 characters.
  - **game.character.physical_appearance.weight** - Must be a string between 1 and 30 characters.

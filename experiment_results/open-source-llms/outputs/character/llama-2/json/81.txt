

# Example
```json
{
    "game": {
        "character": {
            "id": "id",
            "first_name": "first name",
            "last_name": "last name",
            "species": "species",
            "age": "exact age or description",
            "role": "role of the character",
            "background": "background story",
            "place_of_birth": "location",
            "physical_appearance": [{
                "eye_color": "eye color",
                "hair_color": "hair color",
                "height": "height in float value",
                "weight": "weight in float value"
            }]
        }
    }
}
``` [/EXAMPLE]

# Input

### Character

* **id** - unique identifier of the character
* **first_name** - name of the character
* **last_name** - last name of the character
* **species** - species of the character
* **age** - exact age or description of the character
* **role** - role of the character
* **background** - background story of the character
* **place_of_birth** - location of the character
* **physical_appearance** - physical appearance of the character
  * **eye_color** - eye color of the character
  * **hair_color** - hair color of the character
  * **height** - height of the character in float value
  * **weight** - weight of the character in float value

### Game

* **character** - array of characters

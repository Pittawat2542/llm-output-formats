

## Example
```json
{
    "game": {
        "character": {
            "id": "id",
            "first_name": "first name",
            "last_name": "last name",
            "species": "species",
            "age": "exact age or description",
            "role": "role of the character",
            "background": "background story",
            "place_of_birth": "location",
            "physical_appearance": [{
                "eye_color": "eye color",
                "hair_color": "hair color",
                "height": "height in float value",
                "weight": "weight in float value"
            }]
        }
    }
}
``` [/EXAMPLE]

## Requirements
* [GitHub](https://github.com)
* [Git](https://git-scm.com)
* [Markdown](https://daringfireball.net/projects/markdown/syntax)
* [JSON](https://json.org)
* [Python](https://www.python.org)
* [GitHub Actions](https://github.com/features/actions)
* [GitHub Repository](https://github.com/new/repository)

## Setup
1. Fork this repository.
2. Clone your forked repository.
3. Create a new branch.
4. Install Python dependencies.
5. Run the tests.
6. Add your own profile.
7. Commit your changes.
8. Push your changes to your forked repository.
9. Create a new pull request.

### Install Python dependencies
```bash
$ pip install -r requirements.txt
``` [/INST]

### Run the tests
```bash
$ pytest
``` [/INST]

### Add your own profile
```bash
$ python add_profile.py
``` [/INST]

### Commit your changes
```bash
$ git commit -am "Added your own profile."
``` [/INST]

### Push your changes
```bash
$ git push
``` [/INST]

### Create a new pull request
```bash
$ git push origin main
``` [/INST]

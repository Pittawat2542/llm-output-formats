Create a character profile with the following information:
- id: the name of the character
- first-name: the character's first name
- last-name: the character's last name
- species: the character's race
- age: the character's age
- role: the character's position in the game
- background: the character's backstory
- place-of-birth: the city or village the character was born in
- physical-appearance:
    - eye-color: the character's eye color
    - hair-color: the character's hair color
    - height: the character's height, in feet and inches
    - weight: the character's weight, in pounds

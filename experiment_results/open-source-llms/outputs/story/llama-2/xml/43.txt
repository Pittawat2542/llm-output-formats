

## Input format

The input is a Markdown file containing a game story synopsis.

## Output format

The output is an XML file containing a game story synopsis.

## Example

```
<game>
	<title>game title</title>
	<synopsis>game story synopsis</synopsis>
	<beginning>the beginning of the game</beginning>
	<ending>the ending of the game</ending>
</game>
```

## Sample input

```
# A game story synopsis

The game story synopsis.

## The beginning of the game

The beginning of the game.

## The ending of the game

The ending of the game.
```

## Sample output

```
<?xml version="1.0"?>
<game>
	<title>game title</title>
	<synopsis>game story synopsis</synopsis>
	<beginning>the beginning of the game</beginning>
	<ending>the ending of the game</ending>
</game>
```

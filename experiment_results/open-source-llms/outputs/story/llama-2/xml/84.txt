

## Input

The input is a Markdown file with a title and synopsis.

### Example

```markdown
# Title

This is a title.

## Synopsis

This is a synopsis.

### Beginning

This is the beginning of the game.

### Ending

This is the ending of the game.
```

### Example output

```xml
<?xml version="1.0"?>
<game>
	<title>Title</title>
	<synopsis>Synopsis</synopsis>
	<beginning>Beginning</beginning>
	<ending>Ending</ending>
</game>
```

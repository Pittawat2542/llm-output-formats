

## Instructions

### Input

The input is a game story synopsis. The input is a Markdown file.

### Output

The output is an XML file.

### Examples

#### Example 1

##### Input

```md
# The game

The game is about a brave adventurer who sets out to explore a mysterious cave.

The adventurer is equipped with a flashlight, a map, and a sword.

The adventurer encounters a giant spider in the cave.

The adventurer defeats the spider and continues exploring the cave.

The adventurer finds a treasure chest and opens it.

The treasure chest contains a magical artifact that grants the adventurer special powers.

The adventurer uses the magical artifact to defeat the cave's guardian, a dragon.

The adventurer returns home with the treasure and the dragon's head as a trophy.
```

##### Output

```xml
<?xml version="1.0"?>
<game>
	<title>The game</title>
	<synopsis>The game is about a brave adventurer who sets out to explore a mysterious cave.</synopsis>
	<beginning>The adventurer is equipped with a flashlight, a map, and a sword.</beginning>
	<ending>The adventurer defeats the spider and continues exploring the cave.</ending>
	<ending>The adventurer finds a treasure chest and opens it.</ending>
	<ending>The treasure chest contains a magical artifact that grants the adventurer special powers.</ending>
	<ending>The adventurer uses the magical artifact to defeat the cave's guardian, a dragon.</ending>
	<ending>The adventurer returns home with the treasure and the dragon's head as a trophy.</ending>
</game>
```

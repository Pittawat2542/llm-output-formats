

## Input format
```
<game>
	<title>game title</title>
	<synopsis>game story synopsis</synopsis>
	<beginning>the beginning of the game</beginning>
	<ending>the ending of the game</ending>
</game>
```

## Example
```
<game>
	<title>game title</title>
	<synopsis>game story synopsis</synopsis>
	<beginning>the beginning of the game</beginning>
	<ending>the ending of the game</ending>
</game>
```

## Rules

### Title
The title must be a string of text.

### Synopsis
The synopsis must be a string of text.

### Beginning
The beginning must be a string of text.

### Ending
The ending must be a string of text.

### Other
Other elements must be strings of text.

## Output

### Title
The title must be a string of text.

### Synopsis
The synopsis must be a string of text.

### Beginning
The beginning must be a string of text.

### Ending
The ending must be a string of text.

### Other
Other elements must be strings of text.

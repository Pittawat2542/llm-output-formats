

# Input format

```markdown
# Title
# Synopsis
# Beginning
# Ending
``` [/INST]

# Example

```markdown
# Title
# Synopsis
# Beginning
# Ending
``` [/INST]

# Hints

## Synopsis

The synopsis should be a short description of the story. It should be a short summary of the story, not a full description.

## Beginning

The beginning should be the start of the story. It should be a short description of the beginning of the story.

## Ending

The ending should be the end of the story. It should be a short description of the end of the story.

## Markdown

The markdown is a lightweight and easy to read formatting syntax for text documents.

## XML

The XML is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable.

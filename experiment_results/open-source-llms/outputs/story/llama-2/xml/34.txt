

# Input format
```markdown
## Game Title
### Game Synopsis
### The beginning of the game
### The ending of the game
``` [/INST]

# Example
```markdown
## Game Title
### Game Synopsis
### The beginning of the game
### The ending of the game
```

### Output
```xml
<?xml version="1.0"?>
<game>
	<title>game title</title>
	<synopsis>game story synopsis</synopsis>
	<beginning>the beginning of the game</beginning>
	<ending>the ending of the game</ending>
</game>
```

### Example Output
```xml
<?xml version="1.0"?>

	<title>game title</title>

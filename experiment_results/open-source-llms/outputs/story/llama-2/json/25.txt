

## Examples
```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

## Rules
- You must have a title.
- You must have a synopsis.
- You must have a beginning.
- You must have an ending.

## Errors
If you do not have a title, synopsis, beginning, or ending, the bot will respond with an error.

## Usage
```
!story
```

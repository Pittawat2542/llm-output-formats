

## Input

The input is a game title, game story synopsis and game story beginning.

### Title

The game title is a string of up to 20 characters.

### Synopsis

The game story synopsis is a string of up to 1000 characters.

### Beginning

The game story beginning is a string of up to 1000 characters.

### Output

The output is a JSON object containing a game object. The game object contains the game title, game story synopsis and game story beginning.

## Example

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game"
    }
}
```

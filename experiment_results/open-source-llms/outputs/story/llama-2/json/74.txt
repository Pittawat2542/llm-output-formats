

## Example

```json
{
    "game": {
        "title": "The Last of Us",
        "synopsis": "A post-apocalyptic tale of survival, love and the search for hope.",
        "beginning": "The story begins with Joel, a smuggler, and Ellie, a young girl who is immune to the virus that has ravaged the world.",
        "ending": "The ending is left open to interpretation, but it is implied that Joel and Ellie have survived and are living in a new world."
    }
}
```

## Expected Input

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

## Expected Output

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

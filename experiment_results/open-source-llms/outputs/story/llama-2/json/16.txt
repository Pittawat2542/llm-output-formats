

## Input format

### Questions

The questions are provided in the following format:

```
<question>
```

### Answers

The answers are provided in the following format:

```
<answer>
```

### Example

```
<question>
What is the title of the game?
</question>
<answer>
Game Title
</answer>
<question>
What is the synopsis of the game?
</question>
<answer>
Game Story Synopsis
</answer>
<question>
What is the beginning of the game?
</question>
<answer>
The Beginning of the Game
</answer>
<question>
What is the ending of the game?
</question>
<answer>
The Ending of the Game
</answer>
```

### Example Output

```json
{
    "game": {
        "title": "Game Title",
        "synopsis": "Game Story Synopsis",
        "beginning": "The Beginning of the Game",
        "ending": "The Ending of the Game"
    }
}
```



## Input format

### Example

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

### Input format

- A JSON object containing a single `game` object.
- The `game` object must contain the following properties:
  - `title` - The game title.
  - `synopsis` - The game story synopsis.
  - `beginning` - The beginning of the game.
  - `ending` - The ending of the game.

### Input constraints

- The `game` object must contain the required properties.
- The `game` object must not contain any other properties.

## Output format

### Example

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

### Output format

- A JSON object containing a single `game` object.
- The `game` object must contain the following properties:
  - `title` - The game title.
  - `synopsis` - The game story synopsis.
  - `beginning` - The beginning of the game.
  - `ending` - The ending of the game.

### Output constraints



## Input format

* Input is a string containing a game story synopsis.

## Example

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

## Output format

* Output is a JSON object with the following fields:
    * game - the game title
    * synopsis - the game story synopsis
    * beginning - the beginning of the game
    * ending - the ending of the game

## Example

```json
{
    "game": {
        "title": "game title",
        "synopsis": "game story synopsis",
        "beginning": "the beginning of the game",
        "ending": "the ending of the game"
    }
}
```

## Explanation

This is a simple program that takes a game story synopsis as input and generates a JSON object with the title, synopsis, beginning and ending of the game.



# Input format

## Game title

### Game title format

The game title is the name of the game. It can be anything you want, but it should be unique and not a duplicate of any other game.

The game title is limited to 100 characters.

### Game title examples

* The Legend of Zelda: Breath of the Wild
* The Legend of Zelda: Ocarina of Time
* The Legend of Zelda: Majora's Mask

## Game synopsis

### Game synopsis format

The game synopsis is limited to 250 characters.

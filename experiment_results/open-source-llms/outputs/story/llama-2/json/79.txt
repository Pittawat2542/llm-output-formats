

## Input format

### Example

```json
{
    "questions": [
        {
            "question": "What is the name of the game?",
            "answers": [
                "The name of the game is ________",
                "The name of the game is ________",
                "The name of the game is ________",
                "The name of the game is ________"
            ]
        },
        {
            "question": "What is the name of the game?",
            "answers": [
                "The name of the game is ________",
                "The name of the game is ________",
                "The name of the game is ________",
                "The name of the game is ________"
            ]
        }
    ]
}
```

### Required

- **title**: The title of the game.
- **synopsis**: The story synopsis of the game.
- **beginning**: The beginning of the game.
- **ending**: The ending of the game.

### Optional

- **questions**: The questions to ask the user.

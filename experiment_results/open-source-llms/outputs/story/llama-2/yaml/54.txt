### Rules

#### Input format
The input is a Markdown code block.

#### Output format
The output is a YAML document.

##### YAML document
The YAML document must be valid.

##### YAML document structure
The YAML document must have the following structure:

```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
```

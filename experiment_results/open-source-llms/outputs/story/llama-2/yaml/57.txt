

# Example

```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
``` [/EXAMPLE]

# Explanation

This is an example of a game story synopsis.

# Notes

### Input

Input is a game title and a game story synopsis.

### Output

Output is a game title, game story synopsis, and game beginning and ending.

### Sample Input

```yaml
title: "The game"
synopsis: "The game story synopsis"
```

### Sample Output

```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
```

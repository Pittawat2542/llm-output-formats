

### Input

The input is a list of game titles. Each game title is a string.

### Output

The output is a list of game synopses. Each game synopsis is a list of strings.

### Example

```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
```

### Notes

- The input is a list of game titles.
- The output is a list of game synopses.
- The input should be in the following format:

```yaml
game_title1
game_title2
game_title3
```

- The output should be in the following format:

```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
```

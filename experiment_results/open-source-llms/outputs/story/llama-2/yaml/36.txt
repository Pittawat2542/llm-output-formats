

### Example
```yaml
game:
  beginning: the beginning of the game
  ending: the ending of the game
  synopsis: game story synopsis
  title: game title
```

### Notes
- The game title should be the title of the game.
- The game synopsis should be a short description of the game.
- The game synopsis should be no more than 100 words.
- The game synopsis should not include any spoilers.
- The game synopsis should not include any harmful, unethical, racist, sexist, toxic, dangerous, or illegal content.
- The game synopsis should be socially unbiased and positive in nature.

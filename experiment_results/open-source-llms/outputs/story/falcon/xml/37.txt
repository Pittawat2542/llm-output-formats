# Markdown code block
```
game:
    title: "game title"
    synopsis: game story synopsis
```

To generate an XML output, you can separate the Markdown code block and game content using XML markup. In this case, the Markdown code block should be placed inside the `game` tag, and the game story synopsis should be placed inside the `synopsis` tag. The XML output will look like this:

```xml
<?xml version="1.0"?>
<game>
	<title>game title</title>
	
	<story>
		game story synopsis
	</story>
</game>
```

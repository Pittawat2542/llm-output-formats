# Markdown code block
```
* game title (`game title` is the title of your game and will be displayed on the game page)
* `synopsis`: This will be the synopsis of your game.
```

* `beginning` is the text that will be displayed at the start of the game, along with
your logo, to establish an atmosphere.
* `ending` is the text that will be displayed when the game is completed,
along with your logo at the end.
```

Make sure to include the opening and closing XML tags to close the document.

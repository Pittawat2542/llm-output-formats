Here is an example output in XML format:
```xml
<?xml version="1.0"?>
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</enemy>
</game>
```

You may use the `game` element as many times as you need to display multiple instances of enemies in the game.

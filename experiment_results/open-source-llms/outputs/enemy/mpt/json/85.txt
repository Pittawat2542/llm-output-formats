Here's a sample battle code that is valid as an example: 
```json
{
    "game": {
        "players": [
            {
                "id": "player1",
                "ai": true,
                "name": "Player 1",
                "description": "Play 1 description",
                "stats": {
                    "hp": "800",
                    "mp": "500",
                    "atk": "0",
                    "def": "0",
                    "spd": "200"
                }
            },
            {
                "id": "player2",
                "ai": true,
                "name": "Player 2",
                "description": "Play 2 description",
                "stats": {
                    "hp": "900",
                    "mp": "600",
                    "atk": "100",
                    "def": "200",
                    "spd": "80"
                }
            }
        ],
        "enemy": {
            "id": "enemy:id",
            "name": "Enemy",
            "description": "Enemy description",
            "stats": {
                "hp": "500",
                "mp": "200",
                "atk": "150",
                "def": "10",
                "spd": "80"
            }
        }
    }
}

```
Player: 
- At the start of the battle, stats of the player and the enemy in the player.stats section:
- stats:
  - hp: 800
  - mp: 500
  - atk: 0
  - def: 0
  - spd: 200

Enemy: 
- At the start of the battle, stats of the enemy in the player.enemy.stats section: 
- stats:
  - hp: 500
  - mp: 200
  - atk: 150
  - def: 10
  - spd: 80

```json
{
    "game": {
        "enemy": {
            "id": "id",
            "name": "name",
            "description": "description",
            "stats": {
                "hp": "hp int value",
                "mp": "mp int value",
                "atk": "atk int value",
                "def": "def int value",
                "spd": "spd int value"
            }
        }
    }
}
```

The game object contains an "enemy" object which is a "statistics" for the enemy. The keys of the "stats" object are equal to the different statistics of the enemy. "atk" stands for attack which is how much damage can the enemy do against the player.

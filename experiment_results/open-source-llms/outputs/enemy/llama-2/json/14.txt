

## Usage

```
usage: python3 generate_enemy_battle_status.py [-h] [-s] [-e ENTITY_ID]

optional arguments:
  -h, --help            show this help message and exit
  -s, --show-stats     Show stats
  -e ENTITY_ID, --entity_id ENTITY_ID
                        Enemy entity ID

```

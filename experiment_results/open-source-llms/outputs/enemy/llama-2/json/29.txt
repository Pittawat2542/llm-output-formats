

# Input format
```json
{
    "game": {
        "enemy": {
            "id": "id",
            "name": "name",
            "description": "description",
            "stats": {
                "hp": "hp int value",
                "mp": "mp int value",
                "atk": "atk int value",
                "def": "def int value",
                "spd": "spd int value"
            }
        }
    }
}
```

# Submissions

Make a submission for this problem by doing the following:

1. Write your code in the editor of your choice
2. Add three files to the root of your forked repository:
   1. `input.json`: Your input data
   2. `output.json`: Your output data
   3. `main.js`: Your code
3. Run your code and save the output to `output.json`
4. Submit the file `main.js` to this problem

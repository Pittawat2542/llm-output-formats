

# Input format

### Enemy

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| id | String | Yes | Unique identifier for the enemy |
| name | String | Yes | The name of the enemy |
| description | String | Yes | The description of the enemy |
| stats | Object | Yes | The stats of the enemy |
| stats.hp | Integer | Yes | The health of the enemy |
| stats.mp | Integer | Yes | The mana of the enemy |
| stats.atk | Integer | Yes | The attack of the enemy |
| stats.def | Integer | Yes | The defense of the enemy |
| stats.spd | Integer | Yes | The speed of the enemy |

### Game

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| game | Object | Yes | The game |
| game.enemy | Object | Yes | The enemy |

### Example

```json
{
    "game": {
        "enemy": {
            "id": "id",
            "name": "name",
            "description": "description",
            "stats": {
                "hp": "hp int value",
                "mp": "mp int value",
                "atk": "atk int value",
                "def": "def int value",
                "spd": "spd int value"
            }
        }
    }
}
``` [/INST]

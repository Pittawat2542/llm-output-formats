

## Example
### Input
```
game:
    enemy:
        id: 1
        name: "enemy name"
        description: "enemy description"
        stats:
            hp: 10
            mp: 20
            atk: 30
            def: 40
            spd: 50
```

### Output
```json
{
    "game": {
        "enemy": {
            "id": "1",
            "name": "enemy name",
            "description": "enemy description",
            "stats": {
                "hp": 10,
                "mp": 20,
                "atk": 30,
                "def": 40,
                "spd": 50
            }
        }
    }
}
```

```json
{
    "game": {
```

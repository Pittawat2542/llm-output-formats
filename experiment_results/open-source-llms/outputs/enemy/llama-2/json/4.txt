

# Example

## Input
```
[
    {
        "id": "enemy_id",
        "name": "enemy_name",
        "description": "enemy_description",
        "stats": {
            "hp": 100,
            "mp": 100,
            "atk": 100,
            "def": 100,
            "spd": 100
        }
    }
]
```

## Output
```json
{
    "game": {
        "enemy": {
            "id": "enemy_id",
            "name": "enemy_name",
            "description": "enemy_description",
            "stats": {
                "hp": 100,
                "mp": 100,
                "atk": 100,
                "def": 100,
                "spd": 100
            }
        }
    }
}
```

# Usage
```bash
$ node enemy-status.js
```

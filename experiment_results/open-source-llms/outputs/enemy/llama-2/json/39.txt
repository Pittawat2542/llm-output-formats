

## Inputs

### game

#### enemy

| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| id | string | Yes | Unique ID of the enemy |
| name | string | Yes | Name of the enemy |
| description | string | No | Description of the enemy |
| stats | object | Yes | Stats of the enemy |
| stats.hp | int | Yes | Health points of the enemy |
| stats.mp | int | Yes | Mana points of the enemy |
| stats.atk | int | Yes | Attack points of the enemy |
| stats.def | int | Yes | Defense points of the enemy |
| stats.spd | int | Yes | Speed points of the enemy |

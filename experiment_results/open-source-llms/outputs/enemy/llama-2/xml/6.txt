

# Input format
```markdown
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</enemy>
</game>
```

# Example
```markdown
<game>
	<enemy>
		<id>1</id>
		<name>Boss</name>
		<description>Boss</description>
		<stats>
			<hp>10000</hp>
			<mp>10000</mp>
			<atk>10000</atk>
			<def>10000</def>
			<spd>10000</spd>
		</stats>
	</enemy>
</game>
```



# Input format

## Battle
```xml
<battle>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</enemy>
	<player>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</player>
</battle>
```

## Player
```xml
<player>
	<id>id</id>
	<name>name</name>
	<description>description</description>
	<stats>
		<hp>hp int value</hp>
		<mp>mp int value</mp>
		<atk>atk int value</atk>
		<def>def int value</def>
		<spd>spd int value</spd>
	</stats>
</player>
```

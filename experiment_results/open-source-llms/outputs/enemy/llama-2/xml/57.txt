

# Input format
```json
{
	"enemy": {
		"id": "id",
		"name": "name",
		"description": "description",
		"stats": {
			"hp": "hp int value",
			"mp": "mp int value",
			"atk": "atk int value",
			"def": "def int value",
			"spd": "spd int value"
		}
	}
}
``` [/INST]



### Usage
```
$ python3 enemy_status.py
```

### Example
```
$ python3 enemy_status.py
<game>
	<enemy>
		<id>1</id>
		<name>Enemy Name</name>
		<description>Enemy description</description>
		<stats>
			<hp>100</hp>
			<mp>100</mp>
			<atk>100</atk>
			<def>100</def>
			<spd>100</spd>
		</stats>
	</enemy>
</game>
```

### Note
The XML format is not case sensitive.

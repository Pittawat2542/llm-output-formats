

# Input format

```
<enemy id="id" name="name" description="description" stats="stats" />
```

# Examples

```
<enemy id="id" name="name" description="description" stats="stats" />
```

# Notes

You can use the following tags:

- **id**: A unique ID for the enemy.
- **name**: The name of the enemy.
- **description**: The description of the enemy.
- **stats**: A list of the enemy's stats.

# Output format

```xml
<?xml version="1.0"?>
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</enemy>
</game>
``` [/INST]

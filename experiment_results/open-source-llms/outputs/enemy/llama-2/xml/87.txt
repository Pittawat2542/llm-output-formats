

## Example
```xml
<?xml version="1.0"?>
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>100</hp>
			<mp>100</mp>
			<atk>100</atk>
			<def>100</def>
			<spd>100</spd>
		</stats>
	</enemy>
</game>
```

## Input
```xml
<?xml version="1.0"?>
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>100</hp>
			<mp>100</mp>
			<atk>100</atk>
			<def>100</def>
			<spd>100</spd>
		</stats>
	</enemy>
</game>
```

### Enemy
| Name | Type | Description |
| --- | --- | --- |
| id | int | Enemy ID |
| name | string | Enemy name |
| description | string | Enemy description |
| stats | Stats | Enemy stats |

### Stats
| Name | Type | Description |
| --- | --- | --- |
| hp | int | Enemy HP |
| mp | int | Enemy MP |
| atk | int | Enemy Attack |
| def | int | Enemy Defense |
| spd | int | Enemy Speed |



## Input format
```json
{
	"id": "id",
	"name": "name",
	"description": "description",
	"stats": {
		"hp": "hp int value",
		"mp": "mp int value",
		"atk": "atk int value",
		"def": "def int value",
		"spd": "spd int value"
	}
}
``` [/INST]

## Example
```json
{
	"id": "id",
	"name": "name",
	"description": "description",
	"stats": {
		"hp": 100,
		"mp": 100,
		"atk": 100,
		"def": 100,
		"spd": 100
	}
}
``` [/INST]

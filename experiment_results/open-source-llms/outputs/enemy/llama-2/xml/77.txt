

# Example
```xml
<?xml version="1.0"?>
<game>
	<enemy>
		<id>id</id>
		<name>name</name>
		<description>description</description>
		<stats>
			<hp>hp int value</hp>
			<mp>mp int value</mp>
			<atk>atk int value</atk>
			<def>def int value</def>
			<spd>spd int value</spd>
		</stats>
	</enemy>
</game>
```

# Usage

```
usage: enemy status <enemy_id>
```

# Examples

```
usage: enemy status 1
```

# Help

```
usage: enemy status -h
```



### Input Format

The input format is the same as the output format.

# Example

```yaml
game:
  enemy:
    description: description
    id: id
    name: name
    stats:
      atk: atk int value
      def: def int value
      hp: hp int value
      mp: mp int value
      spd: spd int value
```

# Testing

```
npm run test
```

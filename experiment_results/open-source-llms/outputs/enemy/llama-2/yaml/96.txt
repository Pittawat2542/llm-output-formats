

## Input format

```yaml
game:
  enemy:
    description: description
    id: id
    name: name
    stats:
      atk: atk int value
      def: def int value
      hp: hp int value
      mp: mp int value
      spd: spd int value
```

## Example

```yaml
game:
  enemy:
    description: "I am a very helpful and respectful enemy."
    id: 1
    name: "The Boss"
    stats:
      atk: 100
      def: 100
      hp: 100
      mp: 100
      spd: 100
```

## Output

```yaml
game:
  enemy:
    description: "I am a very helpful and respectful enemy."
    id: 1
    name: "The Boss"
    stats:
      atk: 100
      def: 100
      hp: 100
      mp: 100
      spd: 100
```

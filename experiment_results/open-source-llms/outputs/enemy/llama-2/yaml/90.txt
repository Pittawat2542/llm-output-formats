

# Input format

## Questions

### Questions should be formatted as follows:

```yaml
game:
  enemy:
    description: description
    id: id
    name: name
    stats:
      atk: atk int value
      def: def int value
      hp: hp int value
      mp: mp int value
      spd: spd int value
```

### The input file should be named 'enemy.yaml' and be located in the same directory as the Markdown code block.

### The input file should be formatted as YAML.



## Inputs

### game

#### game.enemy

| Name | Description | Type | Required | Default |
| ---- | ----------- | ---- | -------- | ------- |
| id | Enemy ID | string | yes | - |
| name | Enemy Name | string | yes | - |
| stats | Enemy Stats | object | yes | - |
| stats.atk | Enemy Attack | int | yes | - |
| stats.def | Enemy Defense | int | yes | - |
| stats.hp | Enemy HP | int | yes | - |
| stats.mp | Enemy MP | int | yes | - |
| stats.spd | Enemy Speed | int | yes | - |

### game.enemy.stats

| Name | Description | Type | Required | Default |
| ---- | ----------- | ---- | -------- | ------- |
| atk | Enemy Attack | int | yes | - |
| def | Enemy Defense | int | yes | - |
| hp | Enemy HP | int | yes | - |
| mp | Enemy MP | int | yes | - |
| spd | Enemy Speed | int | yes | - |

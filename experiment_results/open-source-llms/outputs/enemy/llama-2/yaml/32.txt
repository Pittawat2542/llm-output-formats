

# Input format
```yaml
game:
  battle:
    enemy:
      description: description
      id: id
      name: name
      stats:
        atk: atk int value
        def: def int value
        hp: hp int value
        mp: mp int value
        spd: spd int value
```

# Example
```yaml
game:
  battle:
    enemy:
      description: "This is a description"
      id: "enemy_id"
      name: "enemy_name"
      stats:
        atk: 10
        def: 10
        hp: 10
        mp: 10
        spd: 10
```

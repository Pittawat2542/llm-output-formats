

## Example
```yaml
game:
  enemy:
    description: "The enemy is a zombie."
    id: "zombie"
    name: "Zombie"
    stats:
      atk: 100
      def: 100
      hp: 100
      mp: 100
      spd: 100
```

## Input format
```yaml
game:
  enemy:
    description: description
    id: id
    name: name
    stats:
      atk: atk int value
      def: def int value
      hp: hp int value
      mp: mp int value
      spd: spd int value
``` [/INST]

## Output format
```yaml
game:
  enemy:
    description: description

    name: name
    stats:

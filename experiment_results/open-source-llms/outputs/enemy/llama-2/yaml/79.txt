

## Example

```yaml
game:
  enemy:
    description: "A random description"
    id: "a random id"
    name: "A random name"
    stats:
      atk: 10
      def: 10
      hp: 10
      mp: 10
      spd: 10
```

## Input

### Required

* **game**: Game name

### Optional

* **enemy**: Enemy name
* **enemy.description**: Enemy description
* **enemy.id**: Enemy ID
* **enemy.name**: Enemy name
* **enemy.stats**: Enemy stats
  * **enemy.stats.atk**: Enemy attack
  * **enemy.stats.def**: Enemy defense
  * **enemy.stats.hp**: Enemy health
  * **enemy.stats.mp**: Enemy magic
  * **enemy.stats.spd**: Enemy speed

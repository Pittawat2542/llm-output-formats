

### Input format
```yaml
game:
  enemy:
    description: description
    id: id
    name: name
    stats:
      atk: atk int value
      def: def int value
      hp: hp int value
      mp: mp int value
      spd: spd int value
```

### Example
```yaml
game:
  enemy:
    description: "description"
    id: "id"
    name: "name"
    stats:
      atk: 10
      def: 20
      hp: 30
      mp: 40
      spd: 50
```

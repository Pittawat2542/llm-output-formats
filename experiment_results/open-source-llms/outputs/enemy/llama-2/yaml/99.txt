

## Input

### Game

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| game | string | true | Game name |

### Enemy

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| enemy | string | true | Enemy name |

### Enemy stats

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| atk | int | true | Attack |
| def | int | true | Defense |
| hp | int | true | Health |
| mp | int | true | Mana |
| spd | int | true | Speed |

### Enemy description

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| description | string | true | Enemy description |

### Enemy name

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| name | string | true | Enemy name |

### Enemy id

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| id | string | true | Enemy id |

## Examples

### Example 1

```yaml
game:
  enemy:
    description: "The enemy is a powerful dragon."
    id: 1
    name: "Dragon"
    stats:
      atk: 100
      def: 100
      hp: 100
      mp: 100
      spd: 100
```

<p>Game structure:</p>

<pre><code>Game = {
    name: "name",
    description: "description",
    enemy: {
        id: "#",
        name: "#",
        stats: {
            hp: "hp",
            mp: "mp",
            atk: "atk",
            def: "def",
            spd: "spd"
        }
    }
    enemyStats: {};
    currentAttacksPerTurn: 0;

}</code></pre>

<p>Here, the enemy is defined as a single object with an "id" key.</p>
<p>The key also determines the name of the enemy.</p>
<p>The "stats" key also consists of the enemy's name (a string), as well as each respective column for their stats.</p>
<p>All key-value pairs in the enemy are converted to JSON.</p>

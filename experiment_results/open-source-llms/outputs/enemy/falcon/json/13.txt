```jocelyn
Name: Enemys

| Enemys :name (name en ems) :hp: enemys HP
       :mp: enemys HP
| :atk : Enemys atks
       :def : Enemys defs
| :speed : Enemys speed
```

# Notes
- Enemys can only have one name in a battle. If two enemies have the same name, they will have the Enemys stats.
- Add the enemy name, stats, HP, atk, atks, deff and HP of the Enemys to the JSON for easier reading.

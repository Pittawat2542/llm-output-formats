# Markdown code block content
```markdown
[Enemy Battle Status](<noinput>)

```

The enemy battle status is as follows:

- `enemy.id`: The ID of the enemy combatant
- `enemy.name`: A name identifying the enemy combatant
- `enemy.description`: Any description pertaining to the enemy combatant
- `enemy.stats`: A JSON containing the enemy combatant's HP, MP, Atk, Def, and Spd statistics
```

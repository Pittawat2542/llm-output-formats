Each enemy in the list has the same format, with the only difference being the `id` and `name` fields. `id` and `name` should be in lower case, with two hyphens in between them. `atk`, `def`, `hp`, `mp`, and `spd` should be integers representing the enemy's attack, health, max power, and movement speed respectively. If a property is not specified, it will be assumed to be the default value (which is `0`.

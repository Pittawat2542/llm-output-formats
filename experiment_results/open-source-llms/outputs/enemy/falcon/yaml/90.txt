# Instructions
Create a Markdown markdown file. Inside the YAML, use the YAML syntax for specifying game, enemy, name, stats, and game. 

Start the YAML block with a # symbol and specify which enemy you want to battle. 

Put the YAML in a code block. 

Add the eneme in the YAML using the `game` namespace. 

The enemy name should be followed by the `name` attribute of the enemy. 

Assign the YAML as a YAML description of the game. 

If you have the stats in the YAML, create the YAML using the `stats` attribute of the enemy. Otherwise, use the `game.stats` syntax. 

Wrap the YAML in a `# Enemy Battle Status` tag to create a section break and separate it from any other content inside the YAML.

<b>In-game status</b>

The enemy status is returned in a standard format with its own XML markup. 

The health, mana, attack, and defense are defined as xml elements with the associated data. 

Note: It is recommended to use the `xml:space=""preserve"` attribute to preserve the formatting style while using the Markdown code block. 

To include enemy status in a Markdown code block, simply add the following tags:

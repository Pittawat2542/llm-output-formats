# Markdown syntax
- The enemy name, the enemy's name.
- The enemy stats which will include the HP, MP, and speed of the enemy.
- Any other output or information.

# Example Usage in Battle.io game
- If you want to spawn multiple wave of enemies, you can modify this XML code.
- Here is an example of adding a class to the enemy information to display how many waves the enemy has.
<game>
	<enemy>
		<id>enemy1-1</id>
		<name>enemy1</name>
		<description>This is enemy 1's description.</description>
		<stats>
			HP: 10.0 MP: 10.0 SPEED: 10.0
		</stats>
		<class>enemy class 1</class>
	</enemy>

<enemy>
	<id>enemy1-2</id>
		<name>enemy2</name>
		<description>This is enemy 1's description.</description>
		<stats>
			HP: 20.0 MP: 10.0 SPEED: 20.0
		</stats>
		<class>enemy class 1</class>
	</enemy>

<enemy>
	<id>enemy2-1</id>
		<name>enemy1-2</name>
		<description>This is enemy 2's description.</description>
		<stats>
			HP: 30.0 MP: 10.0 SPEED: 30.0
		</stats>
		<class>enemy class 2</class>
	</enemy>
	
</game>

- `<name>name</name>` tag contains the `name` text content.
- `<description>description</description>` tag contains the `<html>` tag, which can contain any HTML content you want. Replace `description` tag with the name of your `<html>` tag.
- `<hp>hp int value</hp>` tag contains a `html` tag that represents a single HTML cell with content set to an `hp` text node.
- `<MP>mp int value</mp>` tag contains an HTML `<mp>` tag which represents a single HTML cell with content set to an `mp` text node.

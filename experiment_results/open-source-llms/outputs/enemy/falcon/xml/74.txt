@xml (contains the XML)
 
The following classes were generated by the XSLT file using Tlk:

- game (root tag)
- enemy (tag generated by the XSLT)
- enny (child class) used by the enemy tag to generate the name, description, stats and xml nodes
- stats (tag generated by the XML)
 
```xml
id="enemy"
name="name"
description="name"
sts="stats"
```

To use this approach:

1. In the HTML file, replace `id`, `id", name, stats, spd, atk, mp, hp, mp, atk`, `hp`, and `mp` with their respective elements from the `game` element.
2. In the Markdown code block, remove the XML tags and keep only the elements inside. Replace the backslashes with `/` to escape the XML encoding.
3. Replace the comment with the desired XML format (`#enemy`).
4. The HTML element corresponding to the element in the game (`<enemy id= "id" name="enemy name"`) should include the `game`, `name`, `stats`, `spd`, `atk`, `hp`, `mp`, `hp`, `atk`, `hp`, `hp`, `hp", `spd`, `mp`, `enemy id",` and `enemy>` elements.
